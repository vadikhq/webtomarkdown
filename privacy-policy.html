Privacy Policy for Web to Markdown Extension

Last updated: [Current Date]

Overview

Web to Markdown ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension handles your information.

Information We DON'T Collect

We do not collect, store, or transmit any personal information, including:

- ❌ Personal identification information - ❌ Browsing history or web pages visited - ❌ Content of web pages you convert - ❌ Usage analytics or statistics - ❌ Device information - ❌ Location data - ❌ Cookies or tracking data

How the Extension Works

All processing is done locally in your browser:

- ✅ Web page content is processed entirely within your browser - ✅ Markdown conversion happens on your device only - ✅ No data is sent to external servers - ✅ No network requests are made by the extension - ✅ All converted content remains on your device

Permissions Used

Our extension requests the following permissions, used only for core functionality:

Required Permissions: - activeTab: Access current page content for markdown conversion - downloads: Enable downloading converted markdown files - scripting: Inject conversion scripts into web pages - sidePanel: Display the conversion interface - tabs: Manage tab information for conversion - storage: Save user preferences locally in Chrome - host_permissions (): Access any website for conversion

Why We Need These Permissions: - Content Access: To read and convert web page content to markdown - File Operations: To allow you to download converted markdown files - User Interface: To provide the side panel conversion interface - Local Storage: To remember your preferences between sessions

Data Storage

Local Storage Only: - User preferences are stored locally in Chrome's storage - No data is synchronized or backed up to external servers - You can clear all extension data through Chrome's extension settings

Third-Party Services

We do not use any third-party services: - ❌ No analytics services (Google Analytics, etc.) - ❌ No crash reporting services - ❌ No advertising networks - ❌ No social media integrations - ❌ No external APIs or services

Children's Privacy

Our extension does not knowingly collect any information from children under 13 years of age. The extension is designed to work entirely locally without any data collection.

Changes to Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted in the extension's store listing and within the extension itself. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact Information

If you have any questions about this Privacy Policy, please contact us:

- Email: privacy@webtomarkdown.com - GitHub: [github.com/webtomarkdown/extension/issues]

Compliance

This extension complies with: - ✅ Chrome Web Store Developer Program Policies - ✅ Google's User Data Policy - ✅ GDPR requirements (no data collection) - ✅ CCPA requirements (no data collection)

Your Rights

Since we don't collect any personal data: - You don't need to request data deletion (we have none) - You don't need to request data portability (we have none) - You don't need to opt-out of data collection (we don't collect any)

Simply uninstalling the extension removes all local data.

Summary: Web to Markdown is a privacy-first extension that processes everything locally on your device without collecting, storing, or transmitting any personal information.
